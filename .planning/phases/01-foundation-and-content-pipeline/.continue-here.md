<decisions_made>

- **Frontend**: HTMX + Jinja2 + Tailwind/DaisyUI, no npm/build step (speed over polish)
- **CSS hosting**: Self-hosted static files, not CDN (CDN unreliable on iPhone Safari)
- **Content format**: XML delimiters `<IT>`/`<EN>` not JSON — JSON breaks with multiline markdown
- **Generation strategy**: BackgroundTasks (FastAPI) not asyncio.create_task (unreliable in Gunicorn workers)
- **Polling**: HTMX `hx-trigger="every 2s"` on content fragment, `outerHTML` swap replaces spinner with content
- **Wikipedia**: Wikimedia REST API `/page/summary/{title}` — 3s timeout, optional (never blocks page)
- **Explainer prompt**: 7-section markdown structure, `❓` prefix on exam focus bullets
- **DB path**: `data/osteoprep.db` (not project root)
- **Gunicorn timeout**: 120s (Claude generation can take 30s+)
- **Nginx**: TLS on port 8443 with Tailscale cert, proxies to 127.0.0.1:8080
</decisions_made>
